# ManyPixels 空态插画来源说明

本项目 `LkEmpty` 首版内置空态插画参考 ManyPixels 免费插画图库整理，用于 Lucky UI 组件演示与开源组件默认占位图。

- 来源页面：https://www.manypixels.co/gallery
- 记录日期：2026-05-08
- 用途：数据为空、搜索无结果、网络异常、权限受限、消息为空、购物车为空、收藏为空等空状态占位。
- 处理方式：仅保留有限空态插画能力，不提供素材下载器、素材包或图库竞品能力。

根据 ManyPixels Gallery 页面当前许可说明，插画可用于商业与非商业项目且无需署名，但不得将素材作为独立素材包、图库或竞品分发。若后续 ManyPixels 调整许可，应以其官方网站最新许可为准。
