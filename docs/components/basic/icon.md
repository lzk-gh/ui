# Icon 图标

基于 bootstrap-icons 的图标集封装。

## 基础用法

```vue
<lk-icon name="house" />
<lk-icon name="heart" color="#f00" />
<lk-icon name="bell" size="24" />
```

## Props（节选）

- name: 图标名称（参考项目内置 icon 集合）
- size: 尺寸，单位 px
- color: 颜色
